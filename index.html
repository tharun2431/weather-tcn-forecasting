About this app

This Progressive Web App uses four deep learning models (LSTM, TCN, TCN-LSTM Hybrid, and Stacking Ensemble) trained on the Jena Climate dataset to provide deep learning-powered weather forecasts. Weather data is fetched in real-time from the Open-Meteo API.

Model Architecture & Edge Deployment ⚡

Offline Training (Compute Node): The underlying networks (TCN, LSTM, and Stacking Ensembles) underwent supervised learning over 420,551 hourly observations from the Jena Climate Dataset. The models iteratively optimized their parameter weights to capture complex spatiotemporal dependencies and long-term meteorological patterns.

Online Inference (Edge Node): To achieve zero-latency forecasting without backend infrastructure overhead, the trained LSTM weights were serialized into the ONNX (Open Neural Network Exchange) format. This Progressive Web App intercepts standard REST API calls, isolates the T-24hr sequential vector, and executes a localized forward pass using WebAssembly. This ensures secure, real-time edge computing directly on the client device.
